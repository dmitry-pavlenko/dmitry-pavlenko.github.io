Рацека 3А

  • Опубликовано 29.09.2020
  • by admin
  •  в Описания маршрутов

Подход под маршрут со стоянок Рацека до текеторских ночёвок полтора- два часа.

Есть два варианта начала маршрута.

Можно начать лезть по разрушенным скалам прямо вверх от ночёвок, а можно зайти в кулуар между пиком Рацека и безымянным жандармом и оттуда по явно выраженному ответвлению по простым скалам выйти под большой скальный жандарм на гребне.

Жандарм обходится слева по троечным скалам.

Затем маршрут выходит тоже по троечным скалам на небольшую перемычку в гребне и с неё начинается ключевой участок.

Лезется сначала несколько метров по канту и затем слева от канта гребня.

Протяжённость участка порядка двадцати метров. Лазание четвёрочное.

Далее маршрут выходит на простые двоечно- троечные скалы по которым через две верёвки выход на вершину.

Спуск с вершины возможен по петлям по маршруту 2А до перемычки, либо по анкерным станциям на цепях, первая из которых находится примерно в 25ти метрах ниже вершины.

Чтобы выйти на неё надо пройти вниз по гребню в сторону 3А метров двадцать и оттуда можно пешком слезть на полку на которой слева на большом камне будет дюльферная станция.

Следующая станция находится прямо на перегибе стенки над кулуаром. Необходима осторожность при подходе.

С этой станции один дюльфер до спускового кулуара.

Этот маршрут нельзя назвать полноценной 3А так как он слишком короткий.

Но это отличный вариант для скальных занятий в боевой обстановке.

Требуется стандартный набор скального снаряжения: набор закладок, набор френдов, три станционных петли 120-150см, 5-6 свободных карабинов, 5-6 оттяжек.

 

Тэги: ала-арча, горный гид дмитрий павленко, кыргызстан, описание маршрутов ала-арча
